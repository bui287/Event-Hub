    EventHub - Event Booking Website
A modern event booking platform built with vanilla HTML, CSS, and JavaScript featuring a dark theme with neon accents.

      Features
Browse and filter events by category
Detailed event information with galleries
Secure booking/checkout process
User authentication and profile management
Responsive design for all devices
Local storage for data persistence
Real-time search functionality

     Technologies Used
HTML5: Semantic markup
CSS3: Flexbox, Grid, animations
Vanilla JavaScript: No frameworks
LocalStorage: Client-side data storage
How to Run
Open index.html in your web browser
No installation required
Works offline

     Usage
Browse events on the Events page
Click an event for details
Select tickets and proceed to checkout
Create account or login to manage bookings
